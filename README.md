## 怎么用

每个 skill 是一个独立文件夹，里面有：

- `SKILL.md` — 核心文件，复制到你的 Claude Code 即可使用
- `README.md` — 该 skill 的安装说明

**安装方法**：点进对应 skill 文件夹 → 打开 `SKILL.md` → 点右上角 `Raw` → 复制全部内容 → 保存到你的 `.claude/skills/` 目录下
